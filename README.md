# 📱 Introdução ao React Native com Expo

Este projeto consiste em uma aplicação desenvolvida com **React Native utilizando Expo**, criada para introduzir os conceitos básicos de desenvolvimento de aplicações mobile utilizando **JavaScript e React**.

O projeto contém **exemplos e atividades práticas**, com o objetivo de demonstrar a estrutura de um aplicativo React Native, organização de componentes e utilização de mensagens e módulos dentro da aplicação.

Através dessas atividades, é possível compreender melhor como funciona a construção de interfaces mobile utilizando **React Native** e como o **Expo** facilita o desenvolvimento e execução do aplicativo.

# 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

### React Native
Framework desenvolvido pelo **Meta (Facebook)** que permite criar aplicações mobile utilizando **JavaScript e React**, possibilitando o desenvolvimento para **Android e iOS** com uma única base de código.

### Expo
Ferramenta que simplifica o desenvolvimento com **React Native**, permitindo executar e testar aplicações rapidamente sem a necessidade de configurações complexas do ambiente nativo.

### JavaScript
Linguagem de programação utilizada para desenvolver toda a lógica da aplicação.

### React
Biblioteca JavaScript utilizada para construção de **interfaces baseadas em componentes**, permitindo maior reutilização de código.

# 🧩 Estrutura do Projeto

A aplicação está organizada em uma estrutura simples para facilitar o aprendizado.

## src

A pasta `src` contém o código principal da aplicação, incluindo exemplos e atividades desenvolvidas durante as aulas.

### exemplos

Contém exemplos utilizados para demonstrar conceitos básicos de desenvolvimento com React Native.

Esses exemplos ajudam a compreender:

- estrutura de componentes
- organização de arquivos
- utilização de módulos
- exibição de mensagens na interface

### atividades

Contém atividades práticas propostas durante a aula para reforçar os conceitos aprendidos.

Essas atividades envolvem:

- criação de componentes
- separação de código em módulos
- organização da aplicação em pastas

# ▶️ Como Executar o Projeto

Para executar o projeto, é necessário ter instalado:

- **Node.js**
- **Expo CLI** ou **Expo Go**

Passos para execução:

1. Instalar as dependências do projeto
2. Iniciar o servidor de desenvolvimento com Expo
3. Executar o aplicativo em um dispositivo móvel ou emulador

O Expo permite abrir o aplicativo através de:

- **Android**
- **iOS**
- **Web**
- **Expo Go (no celular)**
